# Shelf

Shelf is a feature rich GUI configuration library minecraft fabric mod.

(1.17.1 or below doesn't compile currently)

To test client, clone this repository and run `./gradlew :26.2:runClient`  
You can open shelf config screen with modmenu screen or running the command `/shelf config`.
