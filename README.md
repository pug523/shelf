# Shelf

Shelf is a feature-rich GUI configuration library for Minecraft Fabric mods.

[![Build Status](https://github.com/pug523/shelf/actions/workflows/ci.yaml/badge.svg)](https://github.com/pug523/shelf/actions)
![Minecraft Version](https://img.shields.io/badge/Minecraft-1.18~26.2-blue)
![Mod Loader](https://img.shields.io/badge/Loader-Fabric-orange)

(1.17.1 or below doesn't compile currently)

To test client, clone this repository and run `./gradlew :26.2:runClient`  
You can open shelf's mod config screen with modmenu screen or running the command `/shelf config`.
